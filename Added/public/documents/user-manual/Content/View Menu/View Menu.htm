View Menu

The view menu is accessed from the Panoptes Main Menu in the top left corner of the window.

This Menu Item... Does this...
Show legend Displays or hides the Legend window, which defines the various event icons.
Show recent events Displays or hides the Recent Events window.
Show alarm List Displays the Outstanding Alarms dialog.
Show lat/long under mouse With a map view displayed, displays or hides lat / long coordinates for the current position of the cursor.
Show Helios statuses Displays or hides the Helios Status dialog, which shows whether a given Helios unit is active and enables you to stop, restart, and set the multiplex duty cycle of the unit.
Close all popup windows Closes all popup windows that are currently open. Note that this only closes separate windows, it does not close open dialoges (sub-windows within another window).
