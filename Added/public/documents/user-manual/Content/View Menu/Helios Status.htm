Helios Status

The dialogue is accessed from the View Menu > Show Helios Statuses

Explanation

Some or all of the options described below may not be available to users without administrative privileges.

This Menu Item... Does this...
Helios Unit View the name assigned to the Helios unit using the New Helios Unit dialog.
Status View the status based on the following:
Running - Online and monitoring fibre lines.
Not Running - Online but not monitoring fibre lines.
Inactive - Based on a status set by the System Admin, no communication with the unit is being done.
No Connection - The Panoptes server can not connect. A portion of the network may be down or the Helios unit may be powered off.
Commands

Select one of the following:
- Start transmitting and receiving light pulses along the fibre line.
- Stop transmitting and receiving light pulses along the fibre line.

- to log into the software that controls the Helios unit.
Recheck Statuses Statuses are updated about every two minutes. Click to force a status update.
