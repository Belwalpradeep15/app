Preferences

To access the preference dialogue go to the Fotech menu () and select Preferences.

Explanation

This Item... Does this...
Home display Select whether the default login page displayed when you launch Panoptes shows a list of fibre lines or a map view.
Units

Select the desired units and decimal precision for distance, velocity, acceleration, and Lat/Long.
Control the rounding precision by specifying the number of values after the decimal. Changing units does not change the units Panoptes uses to write data to the database. If you select units that differ from the database information read from the database will be converted before being displayed. The default units are metres for distance and Celsius for temperature.

Velocity display is used in the Event and Sound Field dialoges.
Colours Select from the following:
Fibre line colour - The colour for fibre lines throughout the system.
Highlight colour - The colour of highlights you select. For example, the velocity overlay on the sound field or the highlight on colour maps.
For details on using the palette, see Selecting colours.
Routes

Select from the following:

Route Colour - The colour of the route, which is a polyline that appears on the map and is differentiated from the fibre.

Marker Spacing/Unit - The interval between marker annotations along the route, and the unit of measure.
