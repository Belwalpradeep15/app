Fotech Menu

The Fotech menu () is accessed from the Panoptes Main Menu in the top left corner of the window.

This Menu Item... Does this...
About Panoptes Displays information about the Panoptes software including the version number and third-party credits.
Report a problem Displays the Submit a Problem Report window. Enter details of a problem that you want to report to Fotech for investigation.
Preferences Displays Preferences dialog where you set parameters for the display.
Print Displays the browser's standard print dialog.
Language Change the displayed language
Log out Logs out the current user and returns to the login screen.
