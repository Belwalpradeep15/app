Setting Panoptes Preferences

Change the initial display users see, the units of measure and unit precision, the of the fibre line and highlights, and if configured, a route polyline.

To set Panoptes preferences

  1. From the Fotech menu (), select Preferences. The Preferences dialog appears.
  2. Specify the view that appears when Panoptes initially displays.
  3. In the Units pane, select the units of measurement for various fields and their decimal precision, where the number specified indicates the number of digits to the right of the decimal place.
  4. In either the Colours or the Routes pane, click the field to display the Colours popup palette and Set the Colours, and then click Close to dismiss the Preferences dialog.
    The Routes map layer is only visible when a file that contains route information in YAML format is uploaded and calibrated using the Admin > Routes menu option.
    These settings only affect the current account and are saved for future sessions.
  5. Press [F5] to refresh open windows so that the new settings appear immediately.
