Fibre Line Cloner

The fibre line cloner dialogue is accessed from the Admin Menu > Panoptes Units.

Explanation

  1. Select up or down arrows to show more or less detail for each of the Fibre Lines and Routes to be uploaded.
  2. Click the Clone button to upload all the Fibre Lines and Routes. Any which are not required can be removed by accessing the Admin > Fibre Lines and Admin > Routes menu items.
