Configuring Alarm Suppression Schedules

Block out periods of time (either on certain days, or on a one-off basis) where events are ignored and alarms are not generated.

To configure alarm suppression schedules

  1. From the Admin menu, click Alarm Suppression Schedules. The Alarm Suppression Schedule dialog appears.
  1. In the Alarm Suppression window, click New Alarm Suppression Schedule () to display the New Alarm Suppression window, which includes a text pane on the left, and a map view on the right.
  2. Define a period during which events will not trigger alarms. This period can span multiple contiguous weeks or a portion of one or more days on an ongoing basis. Time must be in the format 00:00 using a 24-hour clock.
  3. Using the map pane above, click Select Search Area () and then drag the fibre to define the desired region to suppress alarms.
  4. Select the type of event alarms to suppress.
  5. Create an exception window if required, which is a subset of the suppression range defined above. Within this exception window, alerts are enabled.
  6. Click Submit in the bottom-right of the text pane that appears in the above window.
  7. Using the Fotech menu (), click Close Window to return to the Alarm Suppression Schedule window, or click Return to schedule list above the name box that appears in the left pane to return to the list of suppression alarms for the current line.
    You can further refine the suppression schedule in the main Alarm Suppression Schedules window by clicking Edit Suppression Schedule () to the right of the suppression schedule to modify.
  8. Using the Fotech menu (), click Close Window to return to the main window.
