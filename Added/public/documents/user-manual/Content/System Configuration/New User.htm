New User

The new user dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Full name The first and last name for the user account to create. This name appears throughout the application.
Login name A unique name the user types to gain access to the system.
Organizations Select an organization from the drop-down list. Each user must be a member of one or more organizations, which determines what fibre lines they can see.
Click either of the following:
- Add another Organizations drop-down list below the existing one from which you can select another organization.
- Remove the adjacent organization from the user's account profile. You can't remove an organization when only one is associated with the user account, but you can change it.
Roles Select a profile that allows the user to perform certain actions based on the following:


Basic User - View data within their associated organization. Limited ability to change fibre line properties.

Organization Admin - Create and edit users within an organization.

Organization Tech - Modify fibre lines within their associated organization and start and stop Helios units. At least one user per organization should have this permission.

System Admin - Create organizations, users, fibre lines, and so one. Generally, only one user should have this permission.
Authentication Select the Reset Password check box and then type the desired password with which a user logs in to Panoptes.
