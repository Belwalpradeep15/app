New Alarm Suppression

The new alarm suppression dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Name A name by which to identify this schedule in other windows.
Start Date Click Calendar () and using the calendar control, select the day on which to apply the schedule.
Time The time using a 24 hour schedule.
Repeating

Select whether this schedule recurs on future dates, and then select the corresponding day check boxes that appear adjacent to the Repeats On label, and click Calendar () to specify the end date using the Ending On field.
Regions

Click to select a region on the map.
Alerts to Suppress Select the type of alerts to suppress. The choices available are based on those configured for the Helios unit.
Exception Windows Click Create New Exception to create an exception in the above suppression schedule, whereby alerts will not be suppressed.
