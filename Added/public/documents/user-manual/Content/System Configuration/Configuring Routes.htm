Configuring Routes

Routes allow the asset being monitored to be given real world designations, for example "Town A to Town B" or "Pumping station to Terminal". This enables a position on a fibre to be mapped to a position on a route, which is easier for operators to interpret. Panoptes can also be configured to show the direction of travel and relative distance to an event.

For repeat events, route data enables Panoptes to further calculate velocity and acceleration of an event.

Route coordinates can be exported, modified, and imported in YAML format. YAML format can be read as text, and routes can be modified using an ASCII text editor. Third-party applications enable you to convert YAML from and to different language formats such as XML so that it can be used with other applications.

Route information is available in SMS or email notifications, XML notifications, and in alarm pop-up details.

Routes can be configured to display differently than the underlying fibre line.

To export routes

Export an existing path in YAML format in order to modify and then import it.

  1. rom the Admin menu, select Routes. The Routes dialog appears.
  2. Click either or Download as YAML.
  3. Modify the file using either an ASCII text editor, or a third-party application that enables you to modify YAML files.

To import routes

Routes are imported from a YAML file. When you upload a Routes file, Panoptes adds the routes, but does not overwrite existing routes.

  1. From the Admin menu, select Routes. The Routes dialog appears.
  2. Click Upload, click Choose File, and navigate to the desired YAML file that contains the path coordinates with which to supplement or overwrite the existing route paths, and then click Upload.
  3. The Routes - Upload dialog displays check boxes with the indicator, Add to the left of each path. Clear the check box to the left of a path to add the path. The new path will not overwrite a path already in Panoptes.
  4. Click Submit.

To remove a route, click above the route table, or click Delete All at the bottom left of the Route window.
