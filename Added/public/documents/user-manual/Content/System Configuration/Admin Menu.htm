Admin Menu

The admin menu is accessed from the Panoptes Main Menu in the top left corner of the window:

Note: Only administrators see these options.

This Menu Item... Does this...
Server performance Displays the Measuring System Performance dialog with which Fotech developers determine where optimizations can be made.
System identity Displays the Name and ID of each Helios unit assigned to a given Panoptes organization.
Organizations Displays the Administer Organizations dialog with which you create and change company Panoptes accounts.
Users Displays the Administer Users dialog with which you create and change Panoptes user accounts.
Helios units Displays the Administer Helios Units dialog with which you configure Helios units.
Fibre lines Displays the Fibre Lines Admin dialog with which you configure fibre lines.
Routes Displays the Routes dialog with which you upload and download polyline coordinates plotted for a given fibre that then enable you to determine distance and velocity of an event.
Event clearing Displays the Event Clearing Configuration dialog with which you configure how events are removed from the display.
Event types Displays the Event Types dialog where you import custom event images that are also used by a Helios unit that the Panoptes unit is monitoring in order to display those custom event images with events that meet certain parameters you define.
Outstanding alarms Displays the Outstanding Alarms dialog with which you show or hide outstanding alarms.
Alarm configurations Displays the Alarm Configurations dialog with which you set the event criteria which leads to an alarm condition.
Alarm suppression schedules Displays the Alarm Suppression Schedule dialog with which you set the exception period within which alarm conditions are prevented.
Notifications Displays the Notifications dialog with which you set the event criteria which leads to an alarm condition.
Purge historical data Displays the Purge Historical Data dialog with which you set the event criteria which leads to an alarm condition.
Markers Add custom image markers along the fibre. You specify the image and the location where it appears along the fibre. The marker image can be selected from a collection of images you upload and changed at any time for a given location along the fibre.
Watchdog restarts enabled Toggle whether the Helios unit and Panoptes are automatically restarted.
Shut-down Panoptes unit Force the Panoptes unit to close all services.
Download diagnostic logs Download a zip file of diagnostic logs to your client machine that can be used for troubleshooting.
