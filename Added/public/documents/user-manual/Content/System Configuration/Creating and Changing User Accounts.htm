Creating and Changing User Accounts

An organization is a group of users. One or more fibrelines are associated with an organization. Only users who are members of an organization can see the fibre lines assigned to that organization.

While membership in an organization defines what fibre data a user can see, each user is also assigned roles, which define what they can do with the fibre data.

User roles


Basic User - View data within their associated organization. Limited ability to change fibre line properties.

Organization Admin - Create and edit users within an organization.

Organization Tech - Modify fibre lines within their associated organization and start and stop Helios units. At least one user per organization should have this permission.

System Admin - Create organizations, users, fibre lines, and so one. Generally, only one user should have this permission.

Permissions are enforced both in the interface by only displaying the functionality to which a user has access, and also on the Panoptes server so that functionality can't be accessed by manually typing URLs or by attacking the JavaScript layers.

Separate procedures for adding and changing user accounts appear below.

Adding a user

  1. From the Admin menu, select Organizations. The Administer Organizations dialog appears.
  1. In the Employee Count cell of the new organization row, click the hyperlinked number of employees. The Administer Users dialog appears.
  1. Click Add (). The New User dialog appears.
  1. Enter user details, and then click Submit.

Click to add additional users, to edit users, to remove users, or click Return to Organizations.

Changing a user account

  1. From the Admin menu, select Organizations. The Administer Organizations dialog appears.
  1. In the Employee Count cell of the new organization row, click the hyperlinked number of employees. The Administer Users dialog appears.
  1. Click Edit (). The Update User dialog appears.
  1. Select the user details, and then click Submit.

Click to add additional users, to edit users, to remove users, or click Return to Organizations.
