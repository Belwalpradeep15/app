Administer Users

The user administration dialogue is accessed from the Admin Menu.

Some or all of the options described below may not be available to users without administrative privileges.

For this feature... Do this...
ID The default sequential number automatically assigned to the user ID on creation.
Full Name The first and last name for the user. This name appears throughout the application.
Login name A unique name the user types to gain access to the system.
Organization(s) The organization(s) of which the user is a member. Each user must be a member of one or more organizations. This determines what fibre lines they see.
Role(s) The role assigned to the user:


Basic User - View data within their associated organization. Limited ability to change fibre line properties.

Organization Admin - Create and edit users within an organization.

Organization Tech - Modify fibre lines within their associated organization and start and stop Helios units. At least one user per organization should have this permission.

System Admin - Create organizations, users, fibre lines, and so one. Generally, only one user should have this permission.
Commands Select from the following:
- Display the Update User dialog to change the organizations and roles associated with a user.
- Display the New User dialog and create a new user account.
- Remove the adjacent user account. If the user has not created or modified any fibre lines the record is deleted; otherwise it is only marked as deleted.
Caution:Do not delete the admin user account or you will not be able to access the system.
