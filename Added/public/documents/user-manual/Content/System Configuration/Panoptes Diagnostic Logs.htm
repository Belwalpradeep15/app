Panoptes Diagnostic Logs

Diagnostics logs are useful to Fotech customer support when trying to diagnose unexpected behaviour. From the Admin menu, selecting Download diagnostics logs will create a compressed file which can be saved to the local machine. Depending on the browser being used, the file should appear and allow the user to open the Downloads folder to save the file to a location. Once saved the file can be sent to Fotech customer support along with details of the issue to be investigated.
