Configuring Fibre Lines

This section describes how to add and change fibre lines.

Note: Adding a name to a portion of fibre facilitates event tracking. Regions can overlap, and the starting position can be a negative value to cover portions of the fibre line that occur before the active fibre begins.

Adding a fibre line

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. In the bottom-right of the Administer Fibre Lines window, click Add New Fibre ().
    The New Fibre Line dialog appears, where you type the details and then click Submit to return to the Administer Fibre Lines window.

Editing a fibre line

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. Hover the cursor over a given field until it changes to a crosshairs (+), double-click to place the cursor in the field, and then type the desired value.
  3. Specify the following options:
    • Helios Units - Select from one of the units you associated with Panoptes.
    • Channel - Where there is more than one fibre line attached to the Optical unit, specify which one to associate this fibre line with. This is the physical input port on the Optical unit into which the fibre is plugged.
    • Vertical - The type of application or mode of the implementation. For example, a fibre might be allocated to railway detection or perimeter monitoring. The type of vertical is defined during the Helios installation. You can select a specific type to limit the type of event signatures to those of a given type.
    • Zero Point - This should match the Zero Point initially defined for a given fibre line using the Helios unit.
    • Active Length - This should match the Active Length defined for a given fibre line using the Helios unit.
  4. Select from the following options in the Command column:

  5. - Launch an information window showing details about a fibre break along the line. Use this marker to locate and resolve the break, and then remove the fibre break flag.

  6. - Launch a map on and click to display the Fibre Coordinates window where you calibrate fibre lines to maps by adding and changing calibration points.

  7. - Launch a window with which to load a schematic diagram and calibrate fibre lines to schematics.

  8. - Launch the Fibre Regions window with which you define regions along the fibre.

  9. - Launch the Manually Calibrate Map window with which you can upload reference points from a csv or kml file.

  10. Splice - Insert a splice in a fibre line.

  11. - Add a new fibre line using the New Fibre Line dialog box.

  12. - Delete the adjacent fibre line. If no fibre shots or events have been received from this fibre line, the line is deleted from the database and cannot be recovered; otherwise it is hidden from the Panoptes interface and can be recovered by Customer Support.

  13. Enter details for the fibre line, and then click Save.
