Manually Calibrate Fibre Lines

The fibre lines manual calibration dialogue is accessed from the Admin Menu > Fibre Lines

Explanation

This Menu Item... Does this...
Latitude | Longitude The lat/long coordinates that intersect the fibre line.
Distance The first and last distances along the fibre line.
Commands Click and to sort the order according to actual distance along the fibre.
Download as CSV Download the above Lat / Long coordinates in csv format so that they can be managed using an ASCII text or spreadsheet editor. csv or kml values can be imported into Panoptes.
Upload CSV File Click and then browse to a comma separated values file, and then click Upload. This file must have a .csv file name suffix.
Upload KML file Click and then browse to a Keyhole Markup Language file created using Google Earth, and then click Upload.
