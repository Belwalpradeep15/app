Alarm Details

The alarm details dialogue is accessed from the Admin Menu > Outstanding Alarms

For this feature... Do this...
Alarm Details View the alarm details including type, time, and source of the alarm.
Acknowledge Click to display the Add Alarm Response dialog where you can type a comment that appears in the Alarm Details window.
Resolve Alarm (Administrators) Click to display the Add Alarm Response dialog and type a comment that's associated with the alarm and visible using the Admin > Outstanding Alarms menu.
Audio Click to open the Audio Player dialog.
