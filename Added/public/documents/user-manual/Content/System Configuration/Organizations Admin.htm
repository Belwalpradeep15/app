Administer Organisations

The organisation administration dialogue is accessed from the Admin Menu.

Some or all of the options described below may not be available to users without administrative privileges.

For this feature... Do this...
Name The name by which to identify a group of users and the fibre lines they see. This can be a company name, a department, or a geographic area.
Employee Count The number of user accounts associated with this organization using either the New User or the Update User dialog es.
Fibre Lines The number of fibre lines associated with an organization using the New Fibre Line dialog.
Commands

Select from the following:

- Add a new Reference point.
- Display the Organizations dialog and create a new organization.
- Remove the adjacent organization. If no fibre lines are assigned to the organization, its record is deleted from the database; otherwise it is only marked as deleted.
