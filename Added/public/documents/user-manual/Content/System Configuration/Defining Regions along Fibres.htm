Defining Regions along Fibres

Create different regions along the fibre. Each region has an identifying name you assign, as well as start and end points. Start and end points can also be imported and exported in csv format to facilitate management.

Region information is appended to event data sent to Panoptes from the Helios unit for further event detail.

Create regions along a fibre

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. To the right of the desired fibre, click Edit Named Fibre Region (). The Fibre Regions dialog appears.
  3. Do one of the following depending on whether you want to manually create region or maintain them using a .csv file:
    • Manual Creation - Click Add New Fibre () The New Fibre Region dialog appears, where you type a unique name, and start and end position for the region, and then click Save.
    • CSV Management (Download from Panoptes) If there are existing regions in Panoptes that you want to export, modify, and then import back into Panoptes, click Download as CSV. Work with CSV files using a text editor or spreadsheet application.
    • CSV Management (Upload from CSV) Click Choose File to display the Open dialog , navigate to the desired CSV file, and then Open. Using the Panoptes Fibre Region window, click Upload.
