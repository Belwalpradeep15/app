Fibre Lines Admin

The fibre lines admin dialogue is accessed from the Admin Menu.

Explanation

Some or all of the options described below may not be available to users without administrative privileges.

This Menu Item... Does this...
ID View the fibre line's internal ID and Name. This information is used to initially configure the Helios unit.
Name A free-form name by which the fibre line is referred to throughout the application.
Organization Select the organization with which to associate a fibre line. Administrative users can set this to any organization. Other users can only set it to an organization of which they are a member.
Helios Unit Select from the Helios units associated with the Panoptes unit.
Channel Where there is more than one fibre line attached to the Optical unit, specify which one to associate this fibre line with. This is the physical input port on the Optical unit into which the fibre is plugged.
Display type Select whether the fibre line is associated with a map or schematic display.
Vertical The type of application or mode of the implementation. For example, a fibre might be allocated to railway detection or perimeter monitoring. The type of vertical is defined during the Helios installation. You can select a specific type to limit the type of event signatures to those of a given type.
Zero point Enter the distance along the fibre line at which to start monitoring. This point is an offset to avoid measuring right from the Helios unit.
Active length Type the length of the fibre for which to receive a signal. This is used for plotting the fibre line on the display type specified above.
Commands

Select from the following:

- Launch an information window showing details about a fibre break along the line. Use this marker to locate and resolve the break, and then remove the fibre break flag.

- Launch a map and click to display the Fibre Coordinates window where you calibrate fibre lines to maps by adding and changing calibration points.

- Launch a window with which to load a schematic diagram and calibrate fibre lines to schematics.

- Launch the Fibre Regions window with which you define regions along the fibre.

- Launch the Manually Calibrate Map window with which you can upload reference points from a csv or kml file.

Splice - Insert a splice in a fibre line.

- Add a new fibre line using the New Fibre Line dialog box.

- Delete the adjacent fibre line. If no fibre shots or events have been received from this fibre line, the line is deleted from the database and cannot be recovered; otherwise it is hidden from the Panoptes interface and can be recovered by Customer Support.
