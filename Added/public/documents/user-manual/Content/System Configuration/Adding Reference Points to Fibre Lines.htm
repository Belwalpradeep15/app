Adding Reference Points to Fibre Lines

Add reference points to a fibre line in order to determine the distance between a reference point and a given event.

To add reference points to fibre lines

  1. From the Admin menu, select Organizations. The Administer Organizations dialog appears.
  1. Click in the Commands column of the row that contains the Organization for which to add reference points. The Reference Points dialog appears.
  1. Do either of the following:
    • To create a new reference point, click to display the New Reference Point dialog where you type a unique label and lat / long coordinates to specify where the reference point appears on the map, and then click Save.
    • To associate an existing reference point with a schematic, click to display the Reference Point Section Location dialog where you select the desired schematic from the drop-down list at the top of the dialog , and then click a point at which to place the reference point on the schematic and click Save, and then close the dialog. Click in adjacent rows to add additional reference points.

    • To delete a reference point, click next to the item to be deleted.
