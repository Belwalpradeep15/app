Calibrating Fibre Lines to Maps

Calibration specifies reference points along a fibre that tie lat / long coordinates to a distance along the fibre. This calibration facilitates identifying where along the fibre a given event occurred.

The above values can be wholly or partially manually calibrated using Panoptes, or you can import the data into Panoptes from either a kml or a csv file.

The underlying map on which the fibre appears can be either an active Google or Leaflet map, or it can be a static map image. The map view can be used as the primary display or as a secondary display. For example, you might create a map with the start and end points of a deviated well that otherwise has a vertical well bore image as its primary display.

To calibrate fibre lines to online maps

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. Click Calibrate to Map () to the right of the desired fibre. The Map Calibration dialog appears.
  3. Navigate using the Zoom control () to increase the portion of the map that is visible in the window. Drag the desired area to the centre of the map dialog and zoom until the desired extents fit within the map window.
  4. Specify the Lat / Long by clicking Edit () to display the Fibre Coordinates dialog and either type the coordinates or click the map to define them.
  5. Type distance references for each of the points you defined above, and then click Save.
    Fibre control points can be determined by physically tapping the actual fibre line in fixed locations (every 100 metres, for example) while watching the Sound Field dialog on the Helios unit, or by estimating the distance between known positions. Associate a distance with every control point. Drag any control points on the map to adjust them.

To manually calibrate fibre lines to offline maps

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. Click Manually Edit/Upload Calibrations () to the right of the fibre line name to calibrate. The Manually Calibrate Fibre Line dialog appears.
  3. Either manually type lat / long coordinates, or click Choose File and import the lat / long coordinates from a comma-separated values (CSV) or keyhole markup language (KML) file, and then click Upload.
  4. Click Download as CSV to modify and then upload the current lat / long coordinates that are maintained in another application. Even if you already have lat / long values in a CSV file, consider clicking Download as CSV just to verify that your CSV file is structured in the manner that Panoptes will accept.
  5. Enter the start and end distance of the fibre line.
  6. Click to create additional rows, type coordinates, and then click to move the rows to the desired position.
  7. Click Submit.
