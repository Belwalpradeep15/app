Configuring Events

Events differ from alarms in that events are individual disturbances detected by Helios, for example walking. Alarms are triggered when the number and frequency of events at a particular location exceeds predefined limits.

Alarms are associated with an organization and are triggered by either certain notifications, or via the threat accumulation profile. Select whether you are prompted to view alarms upon launching Panoptes. Alarms are written to the Panoptes database regardless of whether alarm notification is selected.

Importing custom event types

Panoptes supports custom event images that you export from Helios and then import into Panoptes. These images are visible in the Panoptes fibre display after you configure an Event alarm using the Helios Admin > Alarm configurations menu.

Note:Your Panoptes account requires both Organization Admin and System Admin privileges, which are enabled using the Admin > Users menu option in order to complete this task.
  1. Using the Helios application, import and then export the desired custom event images using the Helios Admin > Alarm Types menu option. For further instruction on this step, refer to the Helios help documentation.
  2. Using Panoptes, select Admin > Event Types to display the Existing Event Types window.
  3. Click Choose File to display the Open dialog , with which you browse to the .tar file of custom event images that you exported from Helios above, select the .tar file, and then click Open.
  4. Click Upload located at the bottom of the Existing Event Types dialog to import the custom images.

Delete any undesired images by clicking (). Images without the () option are protected by Fotech Solutions Limited or image owner and cannot be deleted.

Configuring event clearing

Specify the number of seconds for which to display an event of a given type after the event is first detected. Limiting the duration that an event flag appears on the fibre display can prevent the fibre display from becoming overly cluttered.

  1. From the Admin menu, select Event Clearing. The Event Clearing dialog appears.
  2. Double-click the right-most cell for the desired fibre line, and type the number of seconds for which to display the event after the event is first detected.
    Note:Because Panoptes checks the Event Clearing list every 5 seconds, the minimum value you can type to clear an event is 5. Typing 0 disables event clearing for a given fibre line.
  3. Select whether the clearing interval defined above is applied. If disabled, events are not cleared.
    Even if the Enabled check box is cleared or 0 is specified above, event details can be retrieved using the Search Criteria dialog.
