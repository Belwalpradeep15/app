Adding and Changing Helios Units

This section describes how to add and configure a Helios unit using the Panoptes interface. If the Helios unit is using threat-driven logging send the latest threat accumulation profile to the Helios unit.

Adding a Helios unit

  1. From the Admin menu, select Helios Units. The Helios Units dialog appears.
  2. Click . The New Helios Unit dialog appears.
  3. Enter to Helios unit details, and then click Submit.

Changing a Helios unit

  1. Launch the Helios web interface with which to interact and configure the Helios unit.
  2. From the Panoptes Admin menu, select Helios Units. The Helios Units dialog appears.
  3. Hover the cursor over a given field until it changes to a crosshairs (+), double-click to place the cursor in the field, and then type the desired value.
  4. Select from the following options in the Commands column:
    - Click to define the Helios unit location on a schematic diagram. See next section for details of this operation.
    - Launch the Helios web interface with which you interact with and configure the Helios unit.
    - Delete the adjacent Helios unit record from the database and release any associated fibre lines.
    - Display the New Helios Unit dialog and add a new Helios unit.

To display Helios Units on schematics

  1. From the Panoptes Admin menu, select Helios Units. The Helios Units dialog appears.
  2. Click Set Helios Locations (). The Helios Section Locations dialog appears.
  3. Select the desired image from the drop-down list, and then either click the desired location to place a Helios Unit on the diagram, or click Clear from diagram to remove it.
  4. Click Save to apply the changes made on the thumbnail above to the actual schematic and then from the Fibres menu, select the fibre line modified above to see the changes.
