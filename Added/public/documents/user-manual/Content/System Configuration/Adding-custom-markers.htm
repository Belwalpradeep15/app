Adding Custom Markers

Add custom image markers along the fibre using the Admin > Markers menu option. You specify the image and the location where it appears along the fibre. The marker image can be selected from a collection of images you upload and changed at any time for a given location along the fibre.

To add custom image markers

  1. Click the Admin > Markers menu option to display the Markers window.
  2. In the Marker Type pane, click to display the Marker Type dialog.
  3. Using the Marker Type dialog , type a name that you use when assigning the marker to a location on the map, click Choose File and select the image file to upload, click Save. Repeat the above step for each of the marker types groups to upload to Panoptes. Any of the above groups can be selected for a specific location along the fibre. In the next step, you define the locations along the fibre at which to display markers.
  4. Add markers to the map using the Admin > Markers window, or using the Map interface (see below).

Markers window

In the Markers pane, click to display the Add New Marker dialog , where you specify a name that appears when you hover over the marker image on the map, a description, and the approximate lat long for the image marker along the fibre.
If this process has already been completed for another marker, click the Lat Long fields and Copy-Paste the lat long values into the Lat Long fields in the Add New Marker dialog for simplicity.

From the Type drop-down list, select the Marker Type you created above, and then click Save.
Any marker type added previously using the Marker Types pane can be selected in the drop-down list at any time.

To change the position of the marke see the Map Interface process below.

Map Interface

Click Calibrate to Map () to display the map in a dialog and navigate to the fibre line.

Click Edit Mode () in the top-right of the map window, click a location along the fibre where to place the custom marker.and drag the custom marker to the desired location along the fibre.

The Add New Marker dialog appears with a placement line indicating where the marker will appear along the fibre line.

Specify a name, which appears when you hover the cursor over the marker in the map, a description, and from the Type drop-down list, select the Marker Type you created above, and then click Save.
Any marker type added previously using the Marker Types pane can be selected in the drop-down list at any time.

Change the position of marker types by clicking Edit Mode () in the top-right of the map view, hovering the cursor over the marker to move until the cursor appears as follows: , and then dragging the marker to the desired position.
