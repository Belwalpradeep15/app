Outstanding Alarms

The outstanding alarms dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Alarm Details ID, Time, Event Type, Region (if regions are defined), Threat level, and Status appear.
Events

Select from the following:

View - Click to view the Alarm Details window.
CSV - Click to download the file in comma-separated values format.
Commands

Select from the following:
- Click to display the Alarm dialog where you view alarm details and either dismiss, acknowledge, or resolve an alarm.
- Click to display the Add Alarm Response dialog where you can add details about the alarm.
- Click to display the Resolve Alarm Response dialog where you add details about the resolution.
- Click to remove an alarm from the alarm list. For fibre break alarms, when the alarm is deleted, it remains visible in the fibre line table until data is purged using the Admin > Purge Historical Data menu option.
Search Alarms Click to display the Search Alarms dialog where you specify the type of alarm, alarm status, and date range for which to search.
Show resolved alarms Display all of the alarms that have been recorded.
