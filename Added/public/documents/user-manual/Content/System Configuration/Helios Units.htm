Helios Units

The Helios units dialogue is accessed from the Admin Menu.

Explanation

Some or all of the options described below may not be available to users without administrative privileges.

This Menu Item... Does this...
ID View the fibre line's internal database ID.
Name Type a free-form name by which the Helios unit is referred to throughout the application.
Serial No. Enter a unique name or unique number, preferably either the name typed above (lower-case and without spaces) or the IP address of the Helios unit.
Hostname Enter the hostname or IP address of the Helios unit.
Port Enter the number of the port to which the Panoptes server sends commands to the Helios unit. The default port is 36832 and should not be changed unless instructed by Customer Support to do so.
Multiplex Channel Count The number of separate fibres attached to a Helios machine. Each fibre is treated separately.
Latitude, Longitude Place a visible marker on the map at the location of the Helios unit.
Active Select whether the Helios unit should be polled to determine whether it is online. When browser clients are logged in to the system, a background process periodically polls the Helios unit. This process can take up to 10 seconds if the Helios unit is behind a firewall and the firewall doesn't send a response. Clear this check box to prevent the Helios unit from being polled.
Commands Select from the following:
- Display the Helios Unit Multiplex Duty Cycle dialog and assign fibre lines to the Helios unit.
- Click to select a schematic uploaded earlier that you use to configure the Helios fibre.
.
- Launch the Helios web interface with which to interact and configure the Helios unit.
- Delete the adjacent Helios unit record from the database and release any associated fibre lines.
- Display the New Helios Unit dialog and add a new Helios unit.
