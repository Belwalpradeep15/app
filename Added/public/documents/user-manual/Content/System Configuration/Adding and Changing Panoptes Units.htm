Adding and Changing Panoptes Units

Use this configuration screen to add Panoptes units in order to create an overview of a large installation.

Fibre Line Cloner

The fibre line cloner allows fibre lines defined at local Panoptes units to be imported into a higher level (Aggregated) Panoptes. The clone will import all fibre lines and routes that have been configured. If some are not required, the user will need to go into the fibre line or routes configuration dialogs and remove the unwanted ones after cloning is complete. If a clone operation is performed from a Panoptes that has already been cloned, the fibre lines and routes will be added a second time. This can be useful where changes have been made at the higher level. In this case the user will need to review the entries in the fibre line and route configuration screens and delete the unwanted entries.

Adding a Panoptes unit

  1. From the Admin menu, select Panoptes Units. The Panoptes Units dialogue appears.
  2. Click (). The New Panoptes Unit dialog appears.
  3. Enter the Panoptes unit details, and then click Submit.

Changing a Panoptes unit

  1. From the Admin menu, select Panoptes Units. The Panoptes Units dialogue appears.
  2. Hover the cursor over a given field until it changes to a crosshairs (+), double-click to place the cursor in the field, and then type the desired value.
  3. Click the Clone button to upload the fibre line and route configurations from the selected Panoptes unit, the Fibre Line Cloner dialog appears.
  4. Select up or down arrows to show more or less detail for each of the Fibre Lines and Routes to be uploaded.
  5. Click the Clone button to upload all the Fibre Lines and Routes. Any which are not required can be removed by accessing the Admin/Fibre Lines and Admin/Routes menu items.
