Inserting Splices

Where a splice is inserted along a fibre line, specify the start position along the fibre, and the length of the splice. Regions and lengths already defined for the fibre are automatically adjusted to account for the splice.

To insert splices

  1. From the Admin menu, select Fibre Lines. The Fibre Lines Admin dialog appears.
  2. Click Splice to the right of the desired fibre. The Insert Splice dialog appears.
  3. Enter the distance along the fibre where the splice begins and the positive or negative value to account for how the fibre has changed
  4. Click Submit to save the changes.

Regions, and lengths already defined for the fibre automatically adjust to account for the splice.
