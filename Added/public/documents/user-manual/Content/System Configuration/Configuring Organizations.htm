Configuring Organizations

An organization is a group of users. One or more fibrelines are associated with an organization. Only users who are members of an organization can see the fibre lines assigned to that organization.

To add an organization

  1. From the Admin menu, select Organizations. The Administer Organizations dialog appears.
  2. Click Add (). The Organization dialog appears.
  3. Enter the organization name, and then click Submit.

The Administer Organizations dialog appears showing the new organization with the Administrator user added by default.

To add employees to an organization

  1. In the Employee Count cell of the new organization row, click 1. The Administer Users dialog appears.
  2. Click Add (). The New User dialog appears.
  3. Enter user details, and then click Submit.

Click to add additional users, to edit users, to remove users, or click Return to Organizations.
