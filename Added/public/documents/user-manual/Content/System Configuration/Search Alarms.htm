Search Alarms

The search alarms dialogue is accessed from Admin Menu. > Outstanding alarms

Explanation

This Menu Item... Does this...
Restrict by ID Specify a unique ID assigned to the alarm for which to search.
Restrict by alarm type Select the type of alarm for which to search.
Restrict by threat level Select one or more of the threat levels defined when Configuring Alarms.
Restrict by alarm status Select the alarm status for which to search.
Restrict by date Click the calendar () and select a date, or type a date in the following format: YYYY-MM-DD.
Restrict by route This option allows filtering by the Route classification, if configured. The Route, a specific direction, distance on the route and velocity can all be used to filter alarms.
Order by

Select whether to sort by time, alarm type, or status, and in ascending or descending order.
Include suppressed Specify whether to search for alarms that occurred over a period of time when Panoptes was configured to ignore alarm generation
