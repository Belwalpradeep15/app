Markers

The markers dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Markers  
ID An automatically assigned ID from Panoptes that is generated upon creation of the item.
Name The name that appears in the map dialog when you hover your cursor over the marker. Selected from a drop down list of Marker Types defined by the user in Marker Types.
Description A description to clarify assignment and usage. This field is not displayed on the map.
Lat Long The position of the marker. These fields are automatically updated when the marker is dragged using edit mode () while in the map view.
Type Select from one of the marker types created using the Marker Types pane described below. The image displayed on the map automatically updates to reflect the selection made here.
Commands Select from the following:
- Add a new marker using the Add New Marker dialog.
- Remove an existing marker.
Marker Types  
ID An automatically assigned ID from Panoptes that is generated upon creation of the item.
Name A user defined name entered when the Marker Type is created using the Add New Marker dialog
Icon The image associated with a marker type. This image can be overwritten by adding a marker type with the same name as the existing marker type, but with a different icon.
Commands Select from the following:
- Add a new marker type using the Add New Marker dialog.
- Remove an existing marker type.
