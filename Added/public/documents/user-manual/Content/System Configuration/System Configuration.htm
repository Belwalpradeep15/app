System Configuration

Panoptes system configuration is limited to admin users. If you have admin permissions you need to login from the Default Login page. This will give you access to the admin menu from where the following operations are available:

  • Server performance

  • System identity

  • Manage Organizations

  • Manage Users

  • Manage Helios units

  • Configure fibre lines

  • Configure routes

  • Event clearing

  • View event types

  • Manage outstanding alarms

  • Alarm configurations

  • Alarm suppression schedules

  • Set notifications

  • Purge historical data

  • Set and configure markers

  • Enable watchdog restarts

  • Shut-down Panoptes

  • Download diagnostic logs
