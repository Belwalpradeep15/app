Fibre Regions

The fibre regions dialogue is accessed from the Admin Menu. > Fibre Lines

Explanation

This Menu Item... Does this...
Fibre Regions Click Add New Fibre () to display the New Fibre Region dialog. The Description is a free form text string to allow the user to reference the region. The start and end fields indicate the start and end of the region along the fibre. The only restriction is that values should be within the bounds of the fibre.

Download as CSV

 Download the coordinates of the existing fibre regions to maintain in a third-party tool. These can be uploaded to Panoptes using the Upload Regions option described below.
Upload Regions Upload region coordinates from a csv file into Panoptes by clicking Choose File, navigating to the desired file, and then clicking Upload.
