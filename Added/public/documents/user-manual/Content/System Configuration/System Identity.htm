System Identity

The system identity dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Name The name of the Panoptes unit. This is printed on the map when in portal view
Serial No. (System Identity) The serial number of the Panoptes unit. This number is visible in the > About Panoptes menu.
UUID (System Identity) The UUID code of the Panoptes unit. This number is visible in the > About Panoptes menu. To generate a new UUID press the New button
Custom Name An discretionary name that is printed on the map, when in portal view, to designate this Panoptes
Custom ID A discretionary ID
