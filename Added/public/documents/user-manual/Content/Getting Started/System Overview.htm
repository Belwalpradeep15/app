Fibre optic lines

Fibre optic lines are placed underground along a secure perimeter, along a pipeline or any other asset that needs to be monitored or secured and protected. Pulses of light are directed down the fibre and the reflections received are interpreted by the Helios DAS unit to pinpoint current activity at specific locations along the fibre line.

Helios DAS Unit

The Helios DAS Unit is both an optical transmitter and an optical receiver. It transmits laser light pulses along a fibre line as configured by the user interface. The reflected light received by the unit is used to detect activity along the fibre line. When the Helios unit converts an optical signal to an electrical signal and uses AI software to create activity event reports which it then forwards to the Panoptes server.

Panoptes Server

The Panoptes server acts as both a database of events received from the Helios unit and a web server to which browser clients connect. The Panoptes server forwards details about events that it receives from the Helios unit onto the browser clients while the browser clients are connected. When events received exceed specified criteria, it generates alarms which can alert an operator to take action.

If communication between the Panoptes and Helios units is lost, the Helios unit stores events until communication with the Panoptes unit resumes, and it then forwards those stored events to the Panoptes unit.

At a local level a Panoptes server is configured to communicate with a number of Helios units monitoring multiple fibre optic lines. A Panoptes server can be configured to provide a system level overview across a number of local Panoptes servers allowing a very large system to be monitored from a single browser.

Browser Client

Browser clients can monitor and analyse incoming events during their current session. They can also query the Panoptes server for historic data. Admin accounts can be used to configure settings that affect all Panoptes users.

Redundant Operation

It is possible to configure a pair of Panoptes servers to operate as a redundant pair. This configuration is generally performed by a Fotech engineer. To change this configuration or add a redundant server to an existing system please consult Fotech customer support. The pair of Panoptes communicate with any connected Helios and present a single view of the system to the operator. When operating as a redundant pair, the Panoptes servers will remain in synch and provide seamless handover in the event that the primary server is no longer operational.

In a system which contains an Aggregated Panoptes, the server at the top level will communicate with whichever local server is currently the primary. Any switchovers will be handled automatically by the application software.
