Linking Panoptes Software with a Unit

To use the Panoptes System, it is advised that you attend a Fotech training course to understand how to get the best performance and benefit from the system. If you need to book a training course, contact Customer Support.

There are two way to access Panoptes:

  • Portal View - displays the map, alarms and events.
  • Default Login - allows administrators to configure Panoptes.

Setting up Panoptes (Administrators only)

Below are instructions to start to use the Panoptes System. It is recommended that you read this section before you start. This is not intended as a complete step by step set up guide but as an overview of the steps required to get started.

  1. Make sure the Panoptes server is turned on.
  2. Open your browser and enter the IP address or domain name for the Panoptes server and enter the Administrator username and password when prompted.
  3. Add Helios units to the configuration and verify that communications paths are operational with each unit. The status of each will be displayed, indicating whether communications are operating.
  4. Add fibre lines and routes, linked to Helios units as appropriate.
  5. Add event and alarm types and configure the operation of Panoptes on receipt of events. This includes how events are stored and processed, the thresholds for generating alarms of different types, what suppression schedules are implemented if any and how alarm reports are communicated.
  6. Load any maps or diagrams to be used on the main display window and perform any necessary calibration activities.
  7. Test configuration by creating events such as walking or digging on a fibre and monitoring Panoptes for them to be reported.
