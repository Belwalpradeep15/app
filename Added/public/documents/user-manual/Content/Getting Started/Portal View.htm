Portal view

The default Panoptes display is Portal view. Portal view is a map display showing the connected fibre lines.

On the right of the map is a menu that allows users to manage alarms and monitor the status of connected Helios and Panoptes units, rack temperatures and other hardware.

Outstanding Alarms

Clicking on any of the outstanding alarms or the map icons opens a popup displaying the details of the alarm:

You now have options to acknowledge or resolve the alarm or listen to an audio feed.

Admin Menu

The admin menu at the top of the sidebar has the following options:

  • Resolve Fibre Breaks
  • Resolve all alarms
  • Outstanding alarms

Main Menu

The Main menu () has four options:

  • About
  • Report
  • Language
  • Print

System Health

System health information is at the bottom of the sidebar. It shows the connected units and their current status.
