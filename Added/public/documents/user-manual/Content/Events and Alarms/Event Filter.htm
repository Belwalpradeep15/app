Event Filter

The dialogue is accessed from the Events Menu

Explanation

This Menu Item... Does this...
Restrict to current fibre line This option is only available if you are currently viewing a single fibre line display. Clear this option to perform a historical search on all fibre lines.
Restrict by route This option allows filtering by the Route classification, if configured. The Route, a specific direction, distance on the route and velocity can all be used to filter events.
Restrict event tyles This option allows specific events to be selected for display, the rest being filtered out.
Restrict to spatial filter This option is available if you selected a specific region of fibre on the map using the Select Region () tool. Select this option to ignore events that fall outside of the selected spatial region.
Restrict magnitude, width, or speed Select this option to ignore events that fall outside of the parameter range you define.
