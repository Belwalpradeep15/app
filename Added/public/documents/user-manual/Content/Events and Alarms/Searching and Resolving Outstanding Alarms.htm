Searching and Resolving Outstanding Alarms

Events differ from alarms in that events are individual disturbances detected by Helios, for example walking. Alarms are triggered when the number and frequency of events at a particular location exceeds predefined limits.

Alarms are associated with an organization and are triggered by either certain notifications, or via the threat accumulation profile. Select whether you are prompted to view alarms upon launching Panoptes. Alarms are written to the Panoptes database regardless of whether alarm notification is selected.

To resolve outstanding alarms

  1. From the Admin menu, select Outstanding alarms. The Outstanding Alarms dialog appears.
  2. Click View () to display the Alarm Details dialog, where you can Post the alarm to the main window for further investigation.
  3. Using the Outstanding Alarms dialog, either View (), Acknowledge> () (and investigate) and then Resolve (), or Delete () the alarm.

To search alarms

  1. From the Admin menu, select Outstanding alarms. The Outstanding Alarms dialog appears.
  2. Click Search Alarms at the top of the Outstanding Alarms window. The Search Alarms dialog appears.
  3. Specify the search criteria and the sort order with which to display the results, and then click Submit.
