Event Search

The dialogue is accessed from the Events Menu.

Explanation

This Menu Item... Does this...
Restrict to current fibre line This option is only available if you are currently viewing a single fibre line display. Clear this option to perform a historical search on all fibre lines.
Restrict to spatial filter This option is only available if a spatial filter has been applied on the map view. Select this to apply filtering of events to the selecting region.
Restrict Date/Time Filter between two dates and optional times.
Restrict by ID Search for a particular alarm or event
Restrict by Route This option allows filtering by the Route classification, if configured. The Route, a specific direction, distance on the route and velocity can all be used to filter events.
Restrict event This option allows specific events to be selected for display, the rest being filtered out.
Restrict magnitude, width or speed This allows you to filter the search by event criteria
- Width The length of the fibre affected by an event.
- Velocity The speed at which an event progresses along the fibre.
- Acceleration The speed of an event based on the distance over seconds squared.
- Distance View the distance along the fibre line where the event occurred.
Include suppressed events Search is normally restricted to ignore any events which have been suppressed via other controls. Select this to ignore any other defined suppressions.
Limit number of records To conserve memory, only 100 entries are displayed in the Event Search window. Edit this limit as required
