Investigating Event Details

Fibre responses that fall outside certain ranges are classified as events. These events are written to a database on the Panoptes server, and forwarded to currently connected browser clients.

Multiple events that stem from the same source and occur within a short duration, for example a truck crossing a monitored perimeter, can be grouped as related events. When you search historical events and event tracking is enabled, a expand/contract icon () appears to the left of grouped events that you can click to expand the group and show all grouped events.

Panoptes can be used to view the details of individual events but can also be used to listen to an audio representation of the activity at that location.

To investigate event details

  1. From the View menu, select Show recent events. The Recent Events dialog appears.
  1. Click an event listed in the Recent Events dialog to display the Event dialog , which shows additional event details. You can also click an event flag directly on the geographical map display to launch the Event dialog.
  2. The Event at dialog appears with a call-out line that connects the displayed event to the location on the spatial fibre display as depicted below.

  1. Select from the following:
    • Display - change the current fibre line display to the fibre line display for the current event. This button only appears if the fibre displayed differs from the event for which you are viewing details.
    • Zoom To - zoom in to the event on the map or section diagram.
    • Show Track - highlight a region of the fibre where multiple, related events occurred and to launch the Event Track window, which enables you to click an event to display its details or play () the visual sequence of related events.
    • Listen - open the Audio Player dialog.
