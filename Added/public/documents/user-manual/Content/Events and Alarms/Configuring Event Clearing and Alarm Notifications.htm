Configuring Event Clearing and Alarm Notifications

This section describes how a user can configure how events and alarms are displayed and managed.

The settings allow the user to:

  • Specify the number of seconds for which to display an event of a given type on all of the event windows after the event is first detected.
  • Select whether a prompt is displayed to view alerts when launching Panoptes. Alerts are still written to the Panoptes database regardless of whether alert notification is selected or cleared.

To configure event clearing

  1. From the Admin menu, select Event Clearing. The Event Clearing dialog appears.
  2. Double-click the right-most cell for the desired fibre line, and type the number of seconds for which to display the event on all of the event windows after the event is first detected.

Event windows include the Recent Events, Sound Field, and Time Series windows.

Because Panoptes checks the Event Clearing list once every 5 seconds, the minimum value you can type to clear an event is 5. Typing 0 disables event clearing for a given fibre line.

Even if cleared, event details can be retrieved using the Search Criteria dialog.

To configure alert notifications

  • From the Admin menu, select or clear System alerts enabled.
