Acknowledging Outstanding Alarms

Alarms that haven't yet been resolved appear in the Outstanding Alarms dialog on the main window.

Click to display alarm details and add comments. Administrators can use this dialog to resolve an alarm, or can work with outstanding alarms using the Admin > Outstanding Alarms menu to display the Outstanding Alarms window, which includes additional functionality such as downloading a csv file showing alarm details, batch operations, and deleting an alarm.

Show or hide the Outstanding Alarm dialog using the View menu.

To acknowledge and resolve outstanding alarms

  1. From the View menu, select Show alarm list. The Outstanding Alarms dialog appears.
  2. Click an alarm in the list to display the Alarm Details dialog. You can also click an alarm flag directly on the map or fibre spatial display to launch the Alarm Details dialog.
  3. Click Acknowledge to display the Add Alarm Response dialog and type a comment that appears in the Alarm Details dialog for logging purposes.

Administrators can click Resolve Alarm using the above dialog to display the Add Alarm Response dialog and type a comment associated with the alarm. When the alarm is resolved, it no longer appears in the Outstanding Alarms window, but administrators can view alarm details (and resolve alarms) using the Admin > Outstanding Alarms menu.

To access additional functionality for outstanding alarms

  1. From the Admin menu, select Outstanding Alarms. The Outstanding Alarms popup dialog appears.
  2. Select one or more alarms to work with by selecting the check box(es) in the left of an alarm, and then select from the following options:
  • View () displays the Alarm Details dialog and either Dismiss, Acknowledge (and investigate), or Resolve the alarm. This is the same functionality available in the procedure above.
  • CSV downloads the alarm details in csv format, which includes the alarms time of occurrence, event type, magnitude, width, velocity, and other details.
  • Acknowledge (), Resolve (), or Delete () the alarm directly from the Outstanding Alarms window.
