Viewing Fibre Line Status

The main view when you log onto Panoptes is the Overview List. It shows the connectivity status of all fibre lines to which you have access.

To view fibre line status

  1. From the View menu, select Show Helios statuses. The Helios Status dialog appears.
  1. View the status based on the following:
    • Running - Online and monitoring fibre lines.
    • Not Running - Online but not monitoring fibre lines.
    • Inactive - Based on a status set by the System Admin, no communication with the unit is being done.
    • No Connection - The Panoptes server can not connect. A portion of the network may be down or the Helios unit may be powered off.
  2. Click either of the following:
    , - to start or stop the Helios unit
    , - to start or stop the laser transmission.
    - to log into the software that controls the Helios unit.
    Recheck Statuses - To poll the Helios units for the latest status.
Note: Basic users can stop and start the Helios unit. Admin users can stop and start the Helios unit laser as well as stop and start the Helios unit itself.
