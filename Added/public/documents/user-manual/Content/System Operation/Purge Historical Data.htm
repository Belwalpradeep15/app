Purge Historical Data

The purge historical data dialogue is accessed from the Admin Menu.

Explanation

This Menu Item... Does this...
Events Select to delete events.
Resolved Alarms Select to delete alarm notifications that have been set to a status of resolved. Alarms that have not been flagged as resolved a cannot be deleted.
Remove Data Older Than The number of days after which to delete data.
Begin Purge

Click to start the purge process. You may want to run this process after hours.
