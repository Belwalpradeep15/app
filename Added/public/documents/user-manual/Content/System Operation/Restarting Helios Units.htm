Restarting Helios Units

Panoptes is able to monitor the status of Helios units and can attempt to restart them if a problem is detected.

To automatically restart Helios units

  1. From the Admin menu, make sure the Watchdog Restarts Enabled is selected (signified by a tick on the menu).

  2. From the Admin menu, click Helios Units to display the Helios window, and then click the Launch Standalone Application icon ().
  3. Log into the Helios unit, and then using the Helios Admin menu, click Restart Helios.
