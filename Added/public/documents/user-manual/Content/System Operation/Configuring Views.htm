Configuring Views

The view options are accessed from the Panoptes Menu Bar > View

From the View menu, select from the following:

  • Show Legend - display a legend of event symbols.
  • Show Recent Events - display the Recent Events dialog.
  • Show Alarm List - display the Outstanding Alarms dialog.
  • Show Lat Long Under Mouse - display the lat / long coordinates in a tooltip dialog that hovers near the cursor.
  • Show Helios Statuses - display the Helios Status dialog.

To toggle which map layers appear, click the leaflet (), which appears in the top-right corner of the map, and select Street, Satellite, or Terrain view.
