Viewing Fibre Line Displays (Maps, Schematics, Images)

The links under the Helios Unit/fibre Line column enable you to launch a spatial view of the physical world over which the fibre line is laid. The spatial view can be a map, schematic, or image.

The fibre line is a polygon drawn over the top of the spatial view. The fibre display enables you to see the location along the fibre line where events occur in both historic and real time.

To view fibre displays

  1. Using your browser, navigate to the URL provided by Fotech and log in. The main dialog appears.
  2. By default, the All Lines (List) view appears, but you can change this view to a geographic map view by selecting Fibres > All lines (map) from the main menu, and then navigate to your area of interest. For active maps, you can further select which view to display (satellite, roads, or terrain) by hovering over in the top-right of the map window.
  3. As events are detected, they appear both directly on the map, schematic, or image at the location where they occurred, and in the Recent Events window.
  4. Click an event either on the map or in the Recent Events dialog to show details.
Note:By default, maps use Leaflet, which is an Open Source JavaScript library of inter-active maps. You can specify different map sources such as Googlemaps, which is fee-based.
