Stream audio from data

Stream audio from either live or logged data from a point along the fibre. There are several modes you can use to listen to data from along the fibre: 

  • Live mode - where you specify a location along the fibre and listen to whatever activity is currently underway at that location.
  • Retroactive mode - where you also specify a location along the fibre, but can only listen to data that has been logged for that location.
  • Live mode using a Playback file a second method for streaming audio from logged data is to playback a file logged by Helios, then use the Audio Player in Live mode to listen to the audio. The location along the fibre to listen to is selected in the same way as for Steaming audio from live data.

Playback alarms from either the Alarm or Event Details dialog

Upon viewing an event or alarm in either the Event Details or Alarm dialog, click the Audio button that appears at the bottom-right of the dialog to launch the Audio Player.

The logged data associated with that event or alarm ID is streamed in audio format.

Streaming audio from logged data

Listening to logged data using the Audio Player is referred to as Retroactive mode. Where data is always present for any location along the fibre when in Live mode, when in Retroactive mode data is only present at a location along the fibre if data has been logged for either that specific location or for the overall fibre.
