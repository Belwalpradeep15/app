Event at <time><ID>

The dialogue is accessed by selecting an Event from the Map

Explanation

For this feature... Do this...
Type View the type of event.
Time View the time the event occurred according to the local time zone defined in your browser client.
Magnitude View the maximum magnitude of the event in millivolts.
Distance View the distance along the fibre line where the event occurred and also its real-world position in either Lat/Long (Map), X/Y (schematic), or depth (well bore) depending on the fibre calibration type and lat/long defined for the Helios unit.
Distance The direct distance from the Helios unit, not the linear distance along the fibre.
Velocity The speed at which the event is moving.
Acceleration The increase or decrease between consecutive events.
Display Click to change the All Lines view to the specific map or section diagram on which an event appears.
Zoom to Click to zoom in to the event.
Show Track
(appears if event tracking applied)		 Click to highlight a region of the fibre where multiple, related events occurred and to launch the Event Track window, which enables you to click an event to display its details or play the sequence of related events.
Audio Click to open the Audio Player dialog.
